# **App Name**: TradeFlow

## Core Features:

- Price Tracking: Provide a dashboard displaying current prices for selected cryptocurrencies and US equities.
- Watchlist Management: Allow users to create and manage a watchlist of their preferred cryptocurrencies and US equities.
- News Sentiment Analysis: Utilize a news sentiment analysis tool to generate trading signals based on current news articles, incorporating it into the decision of what/if information should appear.
- Historical Data Visualization: Display historical price charts for cryptocurrencies and US equities using OpenLayers and Chart.js for enhanced visualization.
- Market News Aggregation: Provide a section for displaying general market news and announcements, curated from reputable sources.
- Price Alerts: Allow users to set up price alerts and receive notifications when a cryptocurrency or equity reaches a specific price.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) to convey trust and stability.
- Background color: Very light gray (#F5F5F5) for a clean, uncluttered interface.
- Accent color: Vivid cerulean (#007BA7) to draw attention to important data and interactive elements.
- Body text: 'Inter' (sans-serif) for clear and accessible readability.
- Headline text: 'Space Grotesk' (sans-serif) paired with Inter, adding a modern tech-focused flair.
- Use clean, minimalist icons for navigation and data representation.
- Design a dashboard-style layout that is easily customizable and presents key information clearly.
